The data in this directory are zipped.
Each zip corresponds to an experiment.
Each experiment consists of a set of scenario.
Each scenario consists of a set of instances.
Each instance has:

1. data_in.json: data needed to solve the model.
2. data_out.json: the solution found.
3. options.json: the options used when solving the instance.
4. output.log / results.log: the output log for the solution method.
